### 关于可自定义方面
!!! 注意 首次运行需进行安装 http://localhost:3000/install
- **数据库配置**：在 `install.lock` 中修改数据库配置
- **端口配置**：在 `app.js` 中修改端口配置 
- **样式配置**：在 `public/css/style.css` 中修改您的样式
